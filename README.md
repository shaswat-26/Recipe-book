# 🍲 Recipe Book Web Application

A full-stack Recipe Book web application that allows users to create, manage, and explore recipes in an organized and user-friendly way.

## 🚀 Features

- Add new recipes with details (title, ingredients, instructions)
- View all recipes in a structured layout
- Edit and delete existing recipes
- Responsive UI for better user experience
- Backend API using Node.js and Express.js
- Data storage using MySQL

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MySQL

## 📁 Project Structure
Recipe-book/
│── public/ # Static files (CSS, JS)
│── views/ # HTML or template files
│── routes/ # Express routes
│── models/ # Database logic
│── config/ # DB configuration
│── app.js # Main server file
│── package.json


## ⚙️ Installation & Setup

1. Clone the repository:

git clone https://github.com/shaswat-26/Recipe-book.git

cd Recipe-book


2. Install dependencies:

npm install


3. Setup MySQL database:
- Create a database (e.g., `recipe_db`)
- Import schema (if provided)

4. Configure database connection:
- Update your DB credentials in config file

5. Run the application:

node app.js


6. Open in browser:

http://localhost:3000
